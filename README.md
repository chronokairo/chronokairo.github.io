# ChronoKairo - Site Oficial

Site desenvolvido em React com Vite, hospedado no GitHub Pages.

## Deploy

O deploy é feito automaticamente via GitHub Actions quando há push na branch main.
O workflow busca o código do repositório [chronokairo_site](https://github.com/chronokairo/chronokairo_site) e faz o build para este GitHub Pages.
